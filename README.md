# Netflix Content Analysis Dashboard

## Project Overview

This project is an interactive Power BI dashboard that analyzes Netflix Movies and TV Shows. The dashboard provides insights into content distribution, ratings, genres, directors, countries, and release trends.

The goal of this project is to transform raw Netflix data into meaningful business insights through data cleaning, transformation, visualization, and storytelling.

## Tools Used

* Power BI
* Power Query
* DAX
* CSV Dataset

## Data Cleaning & Transformation

The following data preparation steps were performed:

* Handled missing values in key columns
* Replaced null values in the Director column with "Unknown"
* Corrected data types
* Removed invalid records and errors
* Performed data transformation using Power Query

## Dashboard Features

### Dashboard 1: Netflix Content Overview

* Total Titles KPI
* Total Movies KPI
* Total TV Shows KPI
* Top Countries Analysis
* Rating Distribution
* Top Genres
* Release Trend Analysis
* Business Insights & Recommendations

### Dashboard 2: Content Deep Dive

* Top Directors Analysis
* Rating by Type
* Movie Duration Distribution
* Content by Type Over Years
* Genre Treemap Visualization

## Key Insights

* Movies make up the majority of Netflix content.
* TV-MA is the most common rating category.
* The United States contributes the highest number of titles.
* Content releases increased significantly after 2015.
* Documentaries and Stand-Up Comedy rank among the most popular genres.
